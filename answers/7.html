Closures with example and real life implementation

  • A function along with its lexical scope bundled together is a closure.
  • A closure gives you access to an outer functionâ€™s scope from an inner function
  • Closures are created every time a function is created,at function creation time.
  • Every scope has access to everything outside of its scope
  • First Use case: Factory Functions 
                    

	// Factory Function creating person
	var Person = function (name, age) {
		const greeting = function () {
			return (
				'Hello I am ' + name
					+ '. I am ' + age
					+ ' years old. '
			);
		};
		return greeting;
	};

	var person1 = Person('Abhishek', 20);
	console.log(person1());

	var person2 = Person('Raj', 25);
	console.log(person2());
  • Second Use Case: Private variables in the above example person1.age is undefined
